# 🎀 My Cloudinary Photo Library 🎀

Welcome to my beautiful personal photo gallery\! 🎟️ This project is a digital scrapbook that lets you easily upload, save, and organize all your favorite pictures in the cloud. ☁️💕

Powered by Cloudinary, it makes sure photos load super fast, never get lost, and look absolutely amazing. 🩰💖

## ✨ Features

  * 🖼️ **Beautiful Gallery:** View all your uploaded photos in a neat and clean layout.
  * ☁️ **Cloud Storage:** Safely stores your pictures using Cloudinary so they don't take up space on your computer.
  * 🚀 **Super Fast:** Images are automatically optimized so the website loads in the blink of an eye\!
  * 📱 **Mobile Friendly:** Looks perfectly cute on both mobile phones and desktop screens.
  * 🔄 **Live Updates:** Upload a photo and see it in your gallery instantly.

## 🛠️ Built With

  * **Framework:** [Next.js (v14)](https://nextjs.org/) ⚡
  * **Styling:** [Tailwind CSS](https://tailwindcss.com/) 🎨
  * **Media Storage:** [Cloudinary](https://cloudinary.com/) 📸
  * **Hosting:** [AWS - Amplify](https://main.d3khzzdwa5i772.amplifyapp.com/) 🌐

## 💻 How to Run Locally

Want to run this project on your own computer? Follow these simple steps\! 👩‍💻

**1. Clone the project**

```bash
git clone https://github.com/your-username/Photo-Library.git
cd Photo-Library
```

**2. Install the packages**

```bash
npm install
```

**3. Set up your secret keys**
Create a new file named `.env.local` in the main folder and add your Cloudinary keys like this (no quotation marks\!):

```env
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your_cloud_name
NEXT_PUBLIC_CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

**4. Start the magic\!**

```bash
npm run dev
```

Open [https://main.d3khzzdwa5i772.amplifyapp.com/](https://main.d3khzzdwa5i772.amplifyapp.com/) in your browser to see the app running\! 🌟

## 🚀 Deployment

This project is super easy to deploy for free using **AWS - Amplify**. Just import your GitHub repository to AWS - Amplify, add your three Cloudinary environment variables in the project settings, and click "Deploy"\! 🎉
